About LCS

LCS or Lighting Controlled Simply is a C# desktop based Windows application designed to be an easy to use replacement for two scene preset boards. We are utilizing the open source USB to DMX converter from Enttec to help minimize cost and simplify user hardware complexity. We are looking to deliver all the simplicity and function of a physical board while also having extra tools commonly found in high end boards in order to help the user.

Minimum Viable Product(MVP) Features

  • Live computer control of DMX lights
  • Ability to control any type of DMX functionality including but not limited to Color (RGBCMY), Color Temperature
  • Have instant and timed transitions between at least 2 scenes

Add on Features:

  • Control multiple types of DMX Lights at once
  • Ability to Create timed Show files and a cue list
  • Ability to create light groups of similar but different addressed fixtures
  • Sound activated mode(Control the light via external sound from environment)

Instruction

Our software is developed in Windows System. There may be compatibility issues if you run or developed it in Linux or Mac System.

Hardware Needed:

  • A DMX based light (A light that can be controlled through DMX512)
  • Enttec USB to DMX converter (which can be found here)
  • DMX Cable
  • 5-pin (male) to 3-pin (female) DMX adapter (if necessary for your light)

Software Needed:

  • LCS software
  • Enttec Open DMX USB Driver (which can be found here)

Non-technical User Guide:

  • Properly connect the DMX light and the Enttec Converter to your devices.
  • Download our executable software
  • Install Enttec drivers from Enttec website
  • Double click to run our software (ignore all Windows warnings)
  • Up to this stage, the software should be running, more details and explanations about how to use the software can be found in the non-technical user mannual

Technical User Guide:

If you only want to run this software, please follow the non-technical user guide above. If you want to view the source code or interested in the development of the software, follow the instruction below

  • Properly connect the DMX light and the Enttec Converter to your devices.
  • Download our software (zip file) here, unzip the file. Open the project solution file (LCS\LCS.sln) found in the LCS folder with Visual Studio. Alternatively, one can clone the project directly from our GitHub with Visual Studio
  • Up to this stage, you should have source code in front of you. Please consult the technical documentation
    • for code documentation and more technical details about our project.
